# 🔐 PrivacyMSG

🚀 Стек технологий
Backend: FastAPI
База данных: PostgreSQL
ORM: SQLAlchemy / asyncpg
Аутентификация: JWT (JSON Web Tokens)
Реалтайм: WebSockets
Документация API: Swagger (встроен в FastAPI)

📦 Основной функционал
🔑 Регистрация и авторизация пользователей
🔒 Безопасное хранение паролей (хэширование)
💬 Отправка и получение сообщений в реальном времени
🧑‍🤝‍🧑 Список пользователей и диалогов
📡 WebSocket соединение для мгновенной доставки сообщений
🛡️ JWT-аутентификация
📄 Автоматическая документация API

🔐 Безопасность
Пароли хранятся в захэшированном виде (bcrypt)
Используется JWT для аутентификации
Поддержка защищённого соединения (HTTPS/WSS рекомендуется в продакшене)

💡 Идея проекта

PrivacyMSG создан как учебный и практический проект для изучения:

архитектуры backend-приложений
работы с WebSockets
построения безопасных API
масштабируемых систем обмена сообщениями
